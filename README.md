# Gurun Kit

Vue UI Component Library Built with Tailwind CSS.

[Read Documentation](https://narakode.id/gurunkit).

## Requirements

- Vue.js 3+
- Tailwind CSS 4+

## Installation

```bash
npm install gurunkit
```

## Basic Usage

```vue
<script setup>
import { Button } from 'gurunkit';
</script>

<template>
  <Button>My Button</Button>
</template>
```
